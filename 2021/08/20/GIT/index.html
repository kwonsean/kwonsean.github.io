GIT BASIC!

가장 기초적인 GIT 명령어를 익혀봅시다!

window 에서 git을 사용하는 방법

  • 아래 사이트에 접속하여 다운로드
    Git for Windows
  • 특별한 설정 없이 다운로드 진행
  • git bash 실행

ctrl 과 함께 + or - 를 누르면 글자의 크기를 조절 할 수 있습니다.

ls

  • ls를 입력하면 폴더안에 있는 파일, 폴더들이 보여집니다.
  • ~는 최상위 폴더를 나타냅니다.

ls -a

  • 아까 안보이던 ./ ../ .android/ 같은 파일들이 보이기 시작하는데요.
  • -a를 사용하면 숨겨진 파일도 모두 보여지게 됩니다.

ls -l

  • -l은 파일의 상세 정보를 보여줍니다.
  • 총 파일의 갯수도 알 수 있습니다.

ls -al

  • a와 l을 함께 사용하면 숨겨진 파일의 상세 정보도 알 수 있습니다.
  • total의 수가 올라간걸 볼 수 있습니다.
  • ls -la로 사용해도 같은 결과가 나옵니다.

원하는 폴더로 이동하는 방법

Documents 폴더로 이동해 봅시다.

cd 이동할 폴더명

  • cd (change directory) 이동할폴더명

  • Documents 폴더로 들어왔습니다.

폴더 만들기

Documents폴더 안에 REVIEW라는 새로운 폴더 만들어 봅시다.

mkdir 만들 폴더명

  • mkdir(make directory) 만들 폴더명
  • 입력하면 REVIEW 폴더가 만들어 졌습니다.

cd REVIEW를 이용하여 REVIEW폴더로 이동한 후 ls를 하여 폴더 안을 봅니다.

빈 폴더가 잘 만들어 진것을 확인 할 수 있습니다!


파일 만들기

REVIEW 폴더 안에 파일들을 생성해 봅시다.

touch 파일명.파일확장자

REVIEW 폴더에 test.txt, home.html, qusetion.py, style.css, super.js 파일을 만들었습니다.

다른 폴더에 파일 옮기기

REVIEW 폴더 안에서 another폴더를 만들고 그 안으로 파일을 이동해 봅시다.


mv 옮길파일 옮길폴더

home.html을 another폴더로 mv를 이용하여 이동 시켰습니다.

다른 폴더에 파일 옮기기 심화

  • another폴더 안에서 상위폴더인 REVIEW에 있는 style.css를 가져오는 방법
  • mv ../style.css .

여기서 .은 현재 자신의 위치를 ..은 상위폴더를 가르킵니다!
따라서 ../style.css는 상위폴더에 있는 (REVIEW) style.css를 뜻하고
. 은 현재 위치인 (another폴더)로 옮겨라는 뜻입니다.

  • 그 반대의 경우 즉, 하위 폴더에서 상위폴더로 파일을 옮기는 방법 역시 가능합니다.

파일 이름 변경하기

mv 파일명 바꿀파일명

home.html 이 newHome.html로 변경되었습니다.

파일 삭제하기

rm 삭제할파일

  • rm(remove)를 이용하여 파일을 삭제합니다.

파일 삭제하기 응용

rm \*.삭제하고싶은 확장자

  • 파일명이 무엇이든 지정한 확장자라면 다 삭제합니다.

rm 파일명.\*

  • 어떤 확장자이건 지정한 파일명을 가지고 있다면 다 삭제합니다.

폴더 삭제하기

  • 그냥 rm을 이용해서는 폴더를 삭제할 수 없습니다.

  • 이때 -rf를 이용하면 폴더를 강제 삭제 할 수 있습니다.

이때 폴더 안의 파일들도 모두 다 삭제가 되니 주의하시기 바랍니다.

명령 수행중에 CTRL + C를 누르면 명령이 중지됩니다.

파일 내용 미리 보기

작성한 파일의 내용을 볼 수 있습니다.

cat 파일명

  • cat 파일명을 입력하면 간단하게 파일의 내용을 확인 할 수 있습니다.

BONUS !!

In to the VIM

  • 먼저 README.md를 생성합니다.
  • vim README.md를 입력합니다.

md 파일 이외에도 대부분의 파일이 vim에서 작성이 가능합니다.
다만 vscode에 비해서는 불편함이 있습니다. 그럼에도 VIM을 사용하는 이유는
간단한 편집만 할 경우, vscode같은 에디터가 없는 상황을 대비하기 위해서 라고 생각합니다.
따라서 기본적인 사용 방법을 알아두면 유용하다고 생각합니다.


  • ~은 문서의 마지막을 나타냅니다.

  • vim에서는 마우스를 사용할 수 없습니다.

  • vim에는 총 4개의 모드가 존재합니다.

    1. normal mode (기본 모드)
    2. insert mode (기본 모드에서 i를 누릅니다)
    3. visual mode (기본 모드에서 v를 누릅니다)
    4. command mode (기본 모드에서 :를 누릅니다)

insert, visual, command모드에서 normal모드로 돌아오는 방법은 ESC입니다.
왼쪽 아래에 모드가 나타납니다.

  • vim에 원하는 내용을 입력후에는 normal mode로 돌아와 :wq를 입력하여 나옵니다.
    • w(write) q(quit)
